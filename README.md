# 🎨 QuickDraw Bot

An AI-powered Telegram bot that generates images from text descriptions using Stable Diffusion.

## Features

- 🤖 AI-powered image generation
- 🎯 Simple and intuitive commands
- ⚡ Fast generation with Stable Diffusion 2.1
- 🌐 Deployed on Railway
- 📊 Rate limiting and error handling
- 🎨 Support for multiple AI models

## Commands

| Command | Description |
|---------|-------------|
| `/start` | Welcome message |
| `/help` | Help menu |
| `/about` | Bot information |
| `/generate` | Guided image generation |
| `/cancel` | Cancel generation |
